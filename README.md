# Worlds-Smallest-Sandbox
The Worlds Smallest Sandbox runs on your desktop with sand particles that are smaller than those in real life! *(Depending on what your screen quality is...)*

## Usage
1. Download wss.py.
2. Make sure your environment supports Tkinter.
3. Run wss.py.
4. Left mouse button draws sand, and the right mouse button draws water.


*I recomend a magnifying glass when playing.*
